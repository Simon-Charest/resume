# Resume
A resume created using Node.js.

## Usage

1. Install and configure prerequisites:
```bash
sudo apt install nodejs npm
npm install package.json
npm audit fix --force
sudo ufw allow 3000
sudo ufw enable
```
2. Run Node.js solution:
```
node src/index.js
```
3. Open solution: [http://localhost:3000/](http://localhost:3000/).
